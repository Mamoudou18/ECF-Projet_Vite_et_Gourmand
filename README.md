# ECF-Projet_Vite_-_Gourmand
Dans le cadre de ma formation Développeur Web et mobile, l'objectif de ce projet est de concevoir un site de restauration (TRAITEUR) pour l'entreprise Vite &amp; Gourmand afin de proposer des menus de différents évènements aux visiteurs et ainsi leur permettre de faire une commande depuis l'application.
